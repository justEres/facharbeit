# Mattis-Präsentation

Typst-Präsentation zur Facharbeit von Mattis Becker:

> Cannabiskonsum - Welche neurobiologischen, psychischen und sozialen Folgen kann er im Jugendalter haben?

## Dateien

- `presentation.typ`: Touying-Präsentation mit sichtbaren Stichpunkten und ausführlichen Sprecherkommentaren im Quellcode.
- `sf1_becker_mattis.pdf`: Ausgangsfacharbeit als inhaltliche Grundlage.
- `fonts/` und `vendor/`: lokale Typst-/Schrift-Abhängigkeiten.

## Kompilieren

```sh
typst compile presentation.typ presentation.pdf
```

Die Präsentation ist auf einen ca. 20-minütigen Vortrag ohne fachliches Vorwissen ausgelegt.
